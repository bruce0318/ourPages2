关于我

你好！我是黄卉然，现就读于武汉大学大学地理信息科学专业，对 GIS 可视化与 Web 前端开发充满热情。

我的技术栈包括 Java、Python和Android开发。业余爱好数字绘画和游戏制作，喜欢把地图与艺术结合创作独特的数据可视化作品。

我的技能

下面是我擅长的一些领域。

GIS 可视化

利用 Leaflet、OpenLayers 和 GeoServer，将复杂的空间数据以交互式地图方式呈现。

Android 开发

使用 Java 和 Android Studio，构建高性能、响应式的移动应用。

空间数据分析

使用 Python（GeoPandas、Shapely）对地理数据进行统计、过滤和空间查询。

地图制图设计

善用 QGIS、AI 将地图打造成专业且美观的视觉图件。

自动化脚本

编写Python脚本自动化数据处理、批量任务和部署流程。

数字插画

使用SAI2和PS、AI（滚滚长江东逝水）绘制与创作插画作品，将创意与色彩完美融合。
